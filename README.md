# Minecrap
## Downloading and running the program (mac)
You can download the file directly from here and it will go to your downloads folder.
There are 2 options for running the program:
### From the terminal
Open a new terminal and run the following commands:
`cd Downloads/Minecrap`
`./build/debug/game`
This will open a new terminal window and run the program.
### From Finder
Open finder and click on the "Downloads" folder. Click on the "Minecrap" folder and then "build" and then "debug". In there should be an executable called "game". Double click this and it will run it from a new terminal window.
