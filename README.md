# Minecrap
## Downloading and running the program (mac)
You can download the file directly from here and it will go to your downloads folder.
There are 2 options for running the program:
### From the terminal
Open a new terminal and run the following commands:
```
cd Downloads/Minecrap-main
./build/debug/game
```
This will open a new terminal window and run the program.
### From Finder
Open finder and click on the "Downloads" folder. Click on the "Minecrap-main" folder and then "build" and then "debug". In there should be an executable called "game". Double click this and it will run it from a new terminal window.

### Warning
When using either of these methods, you may get a pop up which warns you that your mac cannot scan this file for malware. To bypass this go to the settings app and then security and privacy. If you scroll down you should see an option to open the executable anyway. If you click on this you may be asked to enter your password, but once you have done so the file should be fine to open.
